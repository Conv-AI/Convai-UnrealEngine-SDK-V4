# Convai Unreal Engine SDK Plugin

## Overview
Convai brings advanced AI-driven interaction to Unreal Engine, empowering NPCs to engage in realistic conversations with players and respond intelligently to verbal commands and environmental events. Convai integrates seamlessly with various character assets and multiple language support.

### Compatibility and Integration
The plugin is compatible with a wide range of character and avatar assets, including Metahumans, Daz3D, Reallusion, ReadyPlayerMe, Avaturn and more. Both humanoid and non-humanoid assets can be seamlessly integrated.

# Installation

1. Download the latest release from [GitHub Releases](https://github.com/Conv-AI/Convai-UnrealEngine-SDK-V4/releases)
2. Extract the downloaded plugin and copy it to one of the following locations:

### For Engine-Level Installation (All Projects)
   - **Windows:**
     ```
     C:\Program Files\Epic Games\UE_5.x\Engine\Plugins\Marketplace
     ```

### For Project-Level Installation (C++ Projects Only)
   - Copy the plugin to your project's `Plugins` folder:
     ```
     YourProject\Plugins\Convai
     ```
   - **Note:** This method requires your project to be a C++ project. If you have a Blueprint-only project, use the engine-level installation method above.

3. Restart Unreal Engine and enable the plugin from `Edit -> Plugins` by searching for "Convai".
